Automated Plantation

I enjoyed the prospect of assisting others with the food created by this invention. However, I rejected this project because plants can grow in suitable conditions even without automation. The project also isn’t specific enough, because different plants have extremely variable tolerances for the variables listed out, and a suitable condition for one can be unsuitable for another.

2D Survival Game

I enjoyed the prospect of creating a game which I could enjoy with others. However, I rejected this project because this concept has been made several times, and with varying themes and ideas. This concept simply feels like less than I can do, and doesn’t push my skills.

DX3

I enjoyed the idea of using this product for myself. However, I rejected this project because the project felt like less than my best effort. DX3 could simply induce the stimulus on a regular interval, and still keep me awake without the use of facial expressions.

Rogue-Like Game

I enjoyed the idea of being able to play my own project. However, I rejected this project because the project has been done before, and to near perfection. I’m also unsure as to whether we will be able to acquire the necessary tools to complete the project.

Restaurant Picker

I enjoyed the prospect of solving disputes between a group on where to eat. However, I rejected this project because there are already several well known free services which display exactly the same details which the restaurant picker will.

The Cafe Project

I enjoyed the prospect of reducing the amount of time required for lunches. However, I rejected this project because there are already several services and applications which have done exactly what this project aims to do.

Tempora

I enjoyed the prospect of the story and narrative that the project was heading towards. However, I rejected this project because the details supplied for the project were vague and could lead to misunderstandings along with an unclear game path, leading to either incompletion or a bad game.

Eagle Card

I enjoyed the prospect of the school modernizing towards the level of a company. However, I rejected this project because the project would never be accepted by Allen High School due to their misplaced funding.

TV360

I enjoyed the prospect of the TV being free to move wherever you would want it to be for convenience. However, I rejected this project because things that are extremely similar to this already exist, and at an affordable price too.

Game Skin Modification

I enjoyed the prospect of saving many people money on in game items. However, I rejected this project because the game itself would fight to counter this as its their main source of revenue for keeping the game running at all.

I enjoyed the prospect of saving energy, and money all while helping the environment with something cheap and convenient. I accepted this project as it seemed at the correct level of CS difficulty, with all of the aforementioned benefits.
